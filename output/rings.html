Notes on Ring Theory and Field Theory

David K. Zhang
Last Modified 2022-03-13

Definition: Domain

A domain is a nontrivial ring in which the product of any two nonzero elements is itself nonzero.

In other words, a domain is a ring that does not contain any (left or right) zero divisors except 000 itself. The most well-studied domains happen to be commutative rings, so there is a special term for a commutative domain.

Definition: Integral Domain

An integral domain is a nontrivial commutative ring in which the product of any two nonzero elements is itself nonzero.

Definition: Principal Ideal, Generator

Let RRR be a commutative ring, and let r∈Rr \in Rr∈R. The principal ideal generated by rrr, denoted by ⟨r⟩\gen{r}⟨r⟩, is the subset of RRR containing all multiples of rrr.

⟨r⟩≔{rx:x∈R} \gen{r} \coloneqq \{ rx : x \in R \} ⟨r⟩:={rx:x∈R}

An ideal I⊴RI \normalin RI⊴R is principal if there exists an element r∈Rr \in Rr∈R such that I=⟨r⟩I = \gen{r}I=⟨r⟩. In this case, we call rrr a generator of III.

Note that a principal ideal may have more than one generator. For example, the ideal of even numbers in Z\ZZ has two generators, ⟨2⟩=⟨−2⟩\gen{2} = \gen{-2}⟨2⟩=⟨−2⟩. In a commutative ring with zero divisors, it can even be the case that a single principal ideal has multiple generators which are not associate to each other.

Two Generators of the Same Ideal in an Integral Domain are Associates

Definition: Principal Ideal Domain (PID)

A principal ideal domain, also known as a PID, is an integral domain in which every ideal is principal.

Definition: Divides, Divisibility Relation, a∣ba \mid ba∣b

Let RRR be a commutative rng, and let a,b∈Ra, b \in Ra,b∈R. We say that aaa divides bbb, denoted by a∣ba \mid ba∣b, if there exists an element q∈Rq \in Rq∈R such that b=qab = qab=qa. This binary relation ∣⊆R×R{\mid} \subseteq R \times R∣⊆R×R is called the divisibility relation on RRR.

The divisibility relation ∣\mid∣ is always reflexive in a ring, but can fail to be reflexive in a rng. Note that 000 does not divide any element of a rng except 000, but every element of a rng divides 000.

Divisibility is Transitive

Theorem: The divisibility relation on any rng is transitive.

Let RRR be a rng, and let a,b,c∈Ra, b, c \in Ra,b,c∈R. If a∣ba \mid ba∣b and b∣cb \mid cb∣c, then there exist x,y∈Rx, y \in Rx,y∈R sch that b=xab = xab=xa and c=ybc = ybc=yb. It follows that c=yxac = yxac=yxa, which proves that a∣ca \mid ca∣c. ∎

Definition: Prime Element

Let RRR be a commutative ring. An element p∈Rp \in Rp∈R is prime if p∉R×∪{0}p \notin R^\times \cup \{0\}p∈/R×∪{0} and for all a,b∈Ra, b \in Ra,b∈R, if p∣abp \mid abp∣ab, then p∣ap \mid ap∣a or p∣bp \mid bp∣b.

The prime elements of the commutative ring Z\ZZ are precisely the usual prime numbers 2,3,5,7,…2, 3, 5, 7, \dots2,3,5,7,… and their negatives.

Definition: Irreducible Element

Let RRR be a ring. An element r∈Rr \in Rr∈R is irreducible if r∉R×r \notin R^\timesr∈/R× and for all a,b∈Ra, b \in Ra,b∈R, if r=abr = abr=ab, then a∈R×a \in R^\timesa∈R× or b∈R×b \in R^\timesb∈R×.

Note that, while the notion of an irreducible element is defined in any ring, the notion of a prime element is only defined in a commutative ring.

Definition: Characteristic, char⁡R\fchar RcharR

Let RRR be a ring. The characteristic of RRR, denoted by char⁡R\fchar RcharR, is the smallest positive integer nnn such that

1R+1R+⋯+1R⏟n copies of 1R=0R. \underbrace{1_R + 1_R + \cdots + 1_R}_{n \text{ copies of } 1_R} = 0_R. n copies of 1R​1R​+1R​+⋯+1R​​​=0R​.

If no such nnn exists, then we define char⁡R≔0\fchar R \coloneqq 0charR:=0.
