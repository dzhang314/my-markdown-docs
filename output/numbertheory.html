Notes on Elementary Number Theory

David K. Zhang
Last Modified 2022-06-01

In these notes, we assume that the reader is familiar with the set of integers, denoted by Z≔{…,−2,−1,0,1,2,… }\Z \coloneqq \{ \dots, -2, -1, 0, 1, 2, \dots \}Z:={…,−2,−1,0,1,2,…}, along with the unary operation of negation −:Z→Z-: \Z \to \Z−:Z→Z and the binary operations of addition and multiplication +,×:Z×Z→Z+, \times: \Z \times \Z \to \Z+,×:Z×Z→Z. As is usual in higher mathematics, we will not explicitly write the multiplication sign ×\times×; instead, we denote the product of xxx and yyy simply by juxtaposition, as in xyxyxy. We will also write x−yx - yx−y to abbreviate the expression x+(−y)x + (-y)x+(−y).

We state without proof the following facts about the arithmetic operations +,−,×+, -, \times+,−,× on Z\ZZ. (Giving full proofs of these statements would require us to work with rigorous, axiomatic definitions of the set Z\ZZ and the functions +++, −-−, and ×\times×, which are surprisingly both difficult to construct and cumbersome to understand. We refer the interested reader to the Wikipedia article on the construction of the integers.)

Elementary Properties of Integer Arithmetic

Theorem: The following statements hold for all a,b,c∈Za, b, c \in \Za,b,c∈Z.

  • Commutative property of addition: a+b=b+aa + b = b + aa+b=b+a.
  • Cancellation property of addition: a+b=a+ca + b = a + ca+b=a+c if and only if b=cb = cb=c.
  • Additive identity property: a+0=aa + 0 = aa+0=a.
  • Additive inverse property: a−a=0a - a = 0a−a=0.
  • Commutative property of multiplication: ab=baa b = b aab=ba.
  • Cancellation property of multiplication: If ab=acab = acab=ac and a≠0a \ne 0a=0, then b=cb = cb=c.
  • Multiplicative identity property: 1a=a1a = a1a=a.
  • Zero-product property: ab=0ab = 0ab=0 if and only if a=0a = 0a=0 or b=0b = 0b=0.
  • Distributive property: a(b+c)=ab+aca (b + c) = ab + aca(b+c)=ab+ac.

Definition: Divisibility, a∣ba \mid ba∣b

Let a,b∈Za, b \in \Za,b∈Z. We say that aaa divides bbb, denoted by a∣ba \mid ba∣b, if there exists an integer c∈Zc \in \Zc∈Z such that b=acb = acb=ac.

Elementary Properties of Divisibility

Theorem: The following statements hold for all k,m,n∈Zk, m, n \in \Zk,m,n∈Z.

  • 1∣n1 \mid n1∣n and −1∣n-1 \mid n−1∣n.
  • 0∣n0 \mid n0∣n if and only if n=0n = 0n=0.
  • If k∣mk \mid mk∣m and k∣nk \mid nk∣n, then k∣m+nk \mid m + nk∣m+n.
  • If k∣mk \mid mk∣m or k∣nk \mid nk∣n, then k∣mnk \mid mnk∣mn.
  • If k∣mk \mid mk∣m and m∣nm \mid nm∣n, then k∣nk \mid nk∣n.

Proof: Let k,m,n∈Zk, m, n \in \Zk,m,n∈Z be given.

  • n=1n=(−1)(−n)n = 1n = (-1)(-n)n=1n=(−1)(−n).
  • 0a=00a = 00a=0 for all a∈Za \in \Za∈Z, so we can write n=0an = 0an=0a for some a∈Za \in \Za∈Z if and only if n=0n = 0n=0.
  • If m=akm = akm=ak and n=bkn = bkn=bk for some a,b∈Za, b \in \Za,b∈Z, then m+n=(a+b)km + n = (a + b)km+n=(a+b)k.
  • If m=akm = akm=ak for some a∈Za \in \Za∈Z, then mn=(an)kmn = (an)kmn=(an)k. Similarly, if n=bkn = bkn=bk for some b∈Zb \in \Zb∈Z, then mn=(mb)kmn = (mb)kmn=(mb)k.
  • If m=akm = akm=ak and n=bmn = bmn=bm for some a,b∈Za, b \in \Za,b∈Z, then n=(ab)kn = (ab)kn=(ab)k. □

Definition: Modular Congruence, x≡y(modk)x \equiv y \pmod{k}x≡y(modk)

Let x,y,k∈Zx, y, k \in \Zx,y,k∈Z. We say that xxx is congruent to yyy modulo kkk, denoted by x≡y(modk)x \equiv y \pmod{k}x≡y(modk), if k∣x−yk \mid x - yk∣x−y.
