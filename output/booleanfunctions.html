Definition: Bit, Bit Vector, Length, Boolean Function, Variables

A Boolean function on n∈Nn \in \Nn∈N variables is a function f:{0,1}n→{0,1}f: \{0, 1\}^n \to \{0, 1\}f:{0,1}n→{0,1} that takes a bit vector of some fixed length nnn and returns a single bit.

Unique Representation of Boolean Functions by Multilinear Polynomials

Theorem: Let FFF be a field, and let n∈Nn \in \Nn∈N. For every Boolean function f:{0,1}n→{0,1}f: \{0, 1\}^n \to \{0, 1\}f:{0,1}n→{0,1}, there exists a unique multilinear polynomial p∈F⌊x1,x2,…,xn⌋p \in F \lfloor x_1, x_2, \dots, x_n \rfloorp∈F⌊x1​,x2​,…,xn​⌋ that represents fff.

Proof: For existence: sum of generalized AND functions.

For uniqueness: suppose there are two and subtract.
