Definition: Alphabet, Letter, Symbol

An alphabet is a finite set. We call the elements of an alphabet letters or symbols.

Definition: Word, Length, Kleene Star, Kleene Closure, Σ∗\Sigma^*Σ∗

A word over an alphabet Σ\SigmaΣ of length n∈Nn \in \Nn∈N is an element of Σn\Sigma^nΣn (the nnn-fold Cartesian product of Σ\SigmaΣ with itself). The set of all words of any length over Σ\SigmaΣ is called the Kleene star or Kleene closure of Σ\SigmaΣ, and is denoted by Σ∗\Sigma^*Σ∗.

In other words, Σ∗≔⋃n∈NΣn\Sigma^* \coloneqq \bigcup_{n \in \N} \Sigma^nΣ∗:=⋃n∈N​Σn, and length defines a function Σ∗→N\Sigma^* \to \NΣ∗→N that maps each element of Σn\Sigma^nΣn to nnn.

Definition: Code, Block Length, Codeword

A code over an alphabet Σ\SigmaΣ of block length n∈Nn \in \Nn∈N is a subset of Σn\Sigma^nΣn. The elements of a code are called its codewords.

Minimum Distance is a Proxy for Robustness

Theorem: Let Σ\SigmaΣ be an alphabet, and let n∈Nn \in \Nn∈N. If a code C⊆ΣnC \subseteq \Sigma^nC⊆Σn has minimum distance ddd, then:

  • CCC can correct ≤d−1\le d - 1≤d−1 erasures.
  • CCC can detect ≤d−1\le d - 1≤d−1 errors.

Definition: Linear Code

Let FFF be a finite field. A linear code of dimension k∈Nk \in \Nk∈N and block length n∈Nn \in \Nn∈N over FFF is a kkk-dimensional linear subspace of FnF^nFn.

Note that the linear-algebraic notion of dimension coincides with our previous definition of the dimension of a code. If CCC is a kkk-dimensional linear subspace of FnF^nFn, then ∣C∣=∣F∣k\abs{C} = \abs{F}^k∣C∣=∣F∣k, which implies k=log⁡∣F∣∣C∣k = \log_{\abs{F}} \abs{C}k=log∣F∣​∣C∣.

Definition: Generator Matrix, Parity-Check Matrix, Dual, C⊥C^\perpC⊥

Let FFF be a finite field, n,k∈Nn, k \in \Nn,k∈N, and let C⊆FnC \subseteq F^nC⊆Fn be a linear code of dimension kkk.

  • A generator matrix for CCC is a matrix G∈Fn×kG \in F^{n \times k}G∈Fn×k whose column space is CCC.
  • A parity-check matrix for CCC is a matrix H∈F(n−k)×kH \in F^{(n - k) \times k}H∈F(n−k)×k whose kernel is CCC.
  • The dual of CCC is the linear code C⊥≔{d∈Fn:⟨c,d⟩=0F for all c∈C}C^\perp \coloneqq \{ \vd \in F^n : \langle \vc, \vd \rangle = 0_F \text{ for all } \vc \in C \}C⊥:={d∈Fn:⟨c,d⟩=0F​ for all c∈C}.

Here, ⟨⋅,⋅⟩\langle \cdot, \cdot \rangle⟨⋅,⋅⟩ denotes the standard inner product on FnF^nFn, i.e., ⟨v,w⟩≔∑i=1nviwi\langle \vv, \vw \rangle \coloneqq \sum_{i=1}^n v_i w_i⟨v,w⟩:=∑i=1n​vi​wi​.

Because every vector space has a basis, every linear code C⊆FnC \subseteq F^nC⊆Fn admits a generator matrix (whose columns are the basis vectors for CCC). In fact, up to permutations of symbol positions within codewords, every linear code admits a generator matrix whose first kkk rows coincide with the k×kk \times kk×k identity matrix. Such a generator matrix defines a systematic encoding map.
