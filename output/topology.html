Notes on Topology

David K. Zhang
Last Modified 2022-03-14

Definition: Topology

Let XXX be a set. A topology on XXX is a collection T⊆P(X)T \subseteq \powerset{X}T⊆P(X) of subsets of XXX that satisfies the following requirements:

  • Empty set is open: ∅∈T\varnothing \in T∅∈T.
  • Whole set is open: X∈TX \in TX∈T.
  • Closed under arbitrary unions: If S⊆TS \subseteq TS⊆T, then ⋃S∈T\bigcup S \in T⋃S∈T.
  • Closed under finite intersections: If S⊆TS \subseteq TS⊆T and ∣S∣\abs{S}∣S∣ is finite, then ⋂S∈T\bigcap S \in T⋂S∈T.

Definition: Topological Space, Point, Open Set

A topological space is an ordered pair (X,T)(X, T)(X,T) consisting of a set XXX, called the underlying set, and a topology TTT on XXX. The elements of XXX are called the points of the topological space (X,T)(X, T)(X,T), and the elements of TTT are called its open sets.

Definition: Closed Set

Let (X,T)(X, T)(X,T) be a topological space. A subset F⊆XF \subseteq XF⊆X is closed if X∖F∈TX \setminus F \in TX∖F∈T.

In other words, a set is closed if and only if its complement is open. Because taking complements is an involution (i.e., X∖(X∖A)=AX \setminus (X \setminus A) = AX∖(X∖A)=A for all A⊆XA \subseteq XA⊆X), the converse is also true: a set is open if and only if its complement is closed.

Note that, in topology, “open” and “closed” are not opposites! It is possible for a set to be both open and closed, and it is also possible for a set to be neither open nor closed. In fact, there is a special name for sets that are both open and closed.

Definition: Clopen Set

Let (X,T)(X, T)(X,T) be a topological space. A subset of XXX is clopen if it is both open and closed.

Every topological space (X,T)(X, T)(X,T) has at least two clopen sets: the empty set ∅\varnothing∅ and the set XXX itself.
