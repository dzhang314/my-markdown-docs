Miscellaneous Thoughts

David K. Zhang
Last Modified 2023-11-02

In these notes, we will develop a generalized form of convex analysis for vector spaces with an arbitrary (possibly infinite) number of dimensions over any ordered field (not just Q\QQ or R\RR).

Definition: Unit Interval, [0F,1F][0_F, 1_F][0F​,1F​], Open Unit Interval, (0F,1F)(0_F, 1_F)(0F​,1F​)

Let FFF be an ordered field. The unit interval in FFF, denoted by [0F,1F][0_F, 1_F][0F​,1F​], is the set

[0F,1F]≔{t∈F:0F≤t≤1F} [0_F, 1_F] \coloneqq \{ t \in F: 0_F \le t \le 1_F \} [0F​,1F​]:={t∈F:0F​≤t≤1F​}

where 0F0_F0F​ and 1F1_F1F​ denote the additive and multiplicative identity elements of FFF, respectively. Similarly, the open unit interval in FFF, denoted by (0F,1F)(0_F, 1_F)(0F​,1F​), is the set

(0F,1F)≔{t∈F:0F<t<1F}. (0_F, 1_F) \coloneqq \{ t \in F: 0_F < t < 1_F \}. (0F​,1F​):={t∈F:0F​<t<1F​}.

Whenever we work in an unspecified ordered field FFF, we will refer to the additive and multiplicative identity elements of FFF as 0F0_F0F​ and 1F1_F1F​, respectively, to remind ourselves that these may not be real numbers.

Definition: Line Segment, [v,w][\vv, \vw][v,w], [v,t,w][\vv, t, \vw][v,t,w]

Let VVV be a vector space over an ordered field FFF. Given two vectors v,w∈V\vv, \vw \in Vv,w∈V, the line segment connecting v\vvv to w\vww is the set

[v,w]≔{tv+(1F−t)w:t∈[0F,1F]}. [\vv, \vw] \coloneqq \{ t \vv + (1_F - t) \vw : t \in [0_F, 1_F] \}. [v,w]:={tv+(1F​−t)w:t∈[0F​,1F​]}.

We also define the notation

[v,t,w]≔tv+(1F−t)w [\vv, t, \vw] \coloneqq t \vv + (1_F - t) \vw [v,t,w]:=tv+(1F​−t)w

to refer to a particular point along this line segment.

Although the notation [v,t,w][\vv, t, \vw][v,t,w] is well-defined for all values of t∈Ft \in Ft∈F, we will only write [v,t,w][\vv, t, \vw][v,t,w] when 0F≤t≤1F0_F \le t \le 1_F0F​≤t≤1F​.

Definition: Convex Set

Let VVV be a vector space over an ordered field FFF. A subset C⊆VC \subseteq VC⊆V is convex if, for all pairs of points v,w∈C\vv, \vw \in Cv,w∈C, the line segment [v,w][\vv, \vw][v,w] is contained in CCC.

It immediately follows from this definition that:

  • Any linear subspace of VVV, including VVV itself, is a convex set.
  • Any singleton subset of VVV is a convex set.
  • The intersection of any number of convex subsets of VVV is a convex set.

Definition: Convex Function

Let VVV be a vector space over an ordered field FFF, and let C⊆VC \subseteq VC⊆V be convex. A function f:C→Ff: C \to Ff:C→F is convex if, for all v,w∈C\vv, \vw \in Cv,w∈C and t∈[0F,1F]t \in [0_F, 1_F]t∈[0F​,1F​], we have

f([v,t,w])≤[f(v),t,f(w)]. f([\vv, t, \vw]) \le [f(\vv), t, f(\vw)]. f([v,t,w])≤[f(v),t,f(w)].

On the right-hand side of this inequality, we regard FFF itself as a vector space over FFF in order to write [f(v),t,f(w)][f(\vv), t, f(\vw)][f(v),t,f(w)]. Note that the domain of a convex function is required to be a convex set in order for f([v,t,w])f([\vv, t, \vw])f([v,t,w]) to be well-defined.

Definition: Strictly Convex Function

Let VVV be a vector space over an ordered field FFF, and let C⊆VC \subseteq VC⊆V be convex. A function f:C→Ff: C \to Ff:C→F is strictly convex if, for all v,w∈C\vv, \vw \in Cv,w∈C and t∈(0F,1F)t \in (0_F, 1_F)t∈(0F​,1F​), we have

f([v,t,w])<[f(v),t,f(w)]. f([\vv, t, \vw]) < [f(\vv), t, f(\vw)]. f([v,t,w])<[f(v),t,f(w)].

Definition: μ\muμ-Strongly Convex Function

Let VVV be an inner product space over an ordered field FFF, let C⊆VC \subseteq VC⊆V be convex, and let μ∈F\mu \in Fμ∈F be positive (μ>0F\mu > 0_Fμ>0F​). A function f:C→Ff: C \to Ff:C→F is μ\muμ-strongly convex if, for all v,w∈C\vv, \vw \in Cv,w∈C and t∈[0F,1F]t \in [0_F, 1_F]t∈[0F​,1F​], we have

f([v,t,w])≤[f(v),t,f(w)]−μ2Ft(1F−t)⟨v−w,v−w⟩. f([\vv, t, \vw]) \le [f(\vv), t, f(\vw)] - \frac{\mu}{2_F} t (1_F - t) \langle \vv - \vw, \vv - \vw \rangle. f([v,t,w])≤[f(v),t,f(w)]−2F​μ​t(1F​−t)⟨v−w,v−w⟩.

Note that division by 2F2_F2F​ is well-defined because an ordered field cannot have positive characteristic.

As a direct corollary of this result, it follows that the squared-norm function x↦⟨x,x⟩\vx \mapsto \langle \vx, \vx \ranglex↦⟨x,x⟩ is 2F2_F2F​-strongly convex, or equivalently, that the scaled function x↦1F2F⟨x,x⟩\vx \mapsto \frac{1_F}{2_F} \langle \vx, \vx \ranglex↦2F​1F​​⟨x,x⟩ is 1F1_F1F​-strongly convex. Definition: Subgradient, Subdifferential, ∂f(v)\partial f(\vv)∂f(v)

Definition: Transpose, gT\vg^TgT, Representable Linear Functional, Representable Dual Space, VTV^TVT

Subdifferentials are Convex Sets

A Hausdorff space is a topological space (X,T)(X, T)(X,T) that has the following property: for all x,y∈Xx, y \in Xx,y∈X, if x≠yx \ne yx=y, then there exist U,V∈TU, V \in TU,V∈T such that x∈Ux \in Ux∈U, y∈Vy \in Vy∈V, and U∩V=∅U \cap V = \varnothingU∩V=∅.

Definition: Basis

Let (X,T)(X, T)(X,T) be a topological space. A basis of (X,T)(X, T)(X,T) is a collection of open sets B⊆TB \subseteq TB⊆T that has the following property: for every open set U∈TU \in TU∈T, there exists a sub-collection C⊆BC \subseteq BC⊆B such that U=⋃CU = \bigcup CU=⋃C.

In other words, a basis is a collection of open sets that allows every open set to be written as a union of basis elements.

Definition: Second-Countable

A topological space is second-countable if it admits a countable basis.

Definition: Manifold, Dimension

A manifold of dimension n∈Nn \in \Nn∈N, also called an n\boldsymbol{n}n-manifold, is a second-countable locally Euclidean Hausdorff space.

If manifolds were not required to be Hausdorff, then the line with two origins would be a one-dimensional manifold. Similarly, if manifolds were not required to be second-countable, then the long line would be a one-dimensional manifold.
